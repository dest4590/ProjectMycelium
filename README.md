<h1 align=center>PROJECT MYCELIUM</h1>

**Project Mycelium** is an advanced OSINT (Open Source Intelligence) and social network analysis tool designed to map,
visualize, and analyze complex digital relationships. It transforms raw social media data into actionable insights by
uncovering hidden connections and identifying community clusters.

The project is named after "Mycelium" (the root structure of fungi) to reflect its ability to branch deep into data
structures and build complex, non-obvious network topologies.

### 👁️ Core Functionalities

- **Recursive Network Mapping:** Implements deep-crawling algorithms to scrape follower/following data across multiple
  levels of depth. This allows for the construction of comprehensive social graphs rather than surface-level snapshots.

- **Temporal Data Tracking:** Features a state-persistent architecture that monitors network evolution. By tracking "
  follow" and "unfollow" events over time, the system provides a historical timeline of social connections.

- **Modular Workspace Management:** Supports project-based data segmentation. Users can categorize targets into distinct
  workspaces (e.g., "Professional," "Industry Specific," "Research Groups") to maintain data integrity and prevent
  cross-contamination of analytical models.

- **Real-time Visualization Dashboard:** Built with a [Vue.js frontend](https://github.com/dest4590/ProjectMycelium-Eye)
  and powered by WebSockets. The dashboard provides live telemetry, streaming processing logs, and dynamic graph
  rendering as data is ingested.

- **Advanced Social Graph Analytics:**
    - **Influence Analysis:** Identifies "key nodes" based on network centrality rather than simple follower counts.
    - **Pathfinding:** Calculates the shortest connection path between two seemingly unrelated profiles.
    - **Community Detection:** Utilizes clustering algorithms to automatically segment the graph into "social circles"
      or organizational units.

### TL;DR

`Mycelium` is a specialized analytical environment for deep social graph research and digital relationship intelligence.
